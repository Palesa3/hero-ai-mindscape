
HERO AI - Mindscape

HERO AI is a functional, highly interactive, and completely operational single-page client-side React web application. It simulates advanced AI intelligence and operations purely on the frontend using reactive state machines—requiring absolutely zero external API keys, backend servers, or operational costs.

## 🎨 Design & Interactive Aesthetics

- **Theme:** Absolute deep space obsidian (`#0B0F19`) accented by glassmorphic panels and backdrop filters.

- **Accents:** Electric neon cyan (`#00F2FE`), toxic neon purple (`#9D4EDD`), and neon success green (`#39FF14`) with custom native Tailwind outer glows.

- **Typography:** Sleek `Orbitron` / `Space Grotesk` headers paired with a crisp, high-density system sans font for content fields.


## 🚀 Core Features & View Controllers

### 1. Advanced State Routing & Navigation

- **Dynamic Content Swapping:** A reactive routing system that toggles application windows instantaneously without refreshing the browser.

- **Desktop Sidebar:** Floating glassmorphic left-side panel with a pulsing logo, active status tracking, and a sticky governance protocol modal trigger.

- **Mobile Layout:** Collapses seamlessly into a sleek bottom neon navigation dock containing matching quick-action icons.


### 2. Operational Modules

* **Mission Control (Dashboard Overview):**

  - *Reactive Drag-and-Drop Dropzone:* Built with native React drag event handlers. Dynamically shifts to a pulsing radar animation upon drag-over and triggers a 1.5-second decryption sequence once a file is dropped.

  - *Quest Automation Tracker:* Global state tracking for "Workplace Tasks Automated Today". Each completed side quest dynamically increments progress by 33%, firing an overlay celebration animation at 100%.

  - *Quick Actions Grid:* Instantly pipelines simulated metrics straight into specialized sub-tools upon selection.


 **Deep Scan Assistant (Research Tool):**

  - *Terminal Execution Output:* Synthesizes inputs into a clean Tactical Summary Card using simulated sequential typing interval delays.

  - *Interactive Data Accordions:* Toggleable structural panels categorizing growth drivers and market friction points.

  - *Interactive Neural Net Map:* A toggleable visual mode mapping text data into connected floating circular nodes with real-time hover tooltips.

**Neural Link (Contextual AI Chatbot):**

  - *Stateful Message Logger:* Pushes user queries and renders realistic AI typing indicators with responsive asynchronous handlers.

  - *Synapse Tuning Widget:* Updates the system's global theme and alters the AI response personality profile dynamically based on selected modes: Tactical (Red/Military), Corporate (Purple/Executive), or Mad Scientist (Toxic Green/Chaotic Jargon).

  - *Floating Command Chips:* Quick-injection pills that feed prompts straight into the communication buffer.

 **Compliance & Governance Protocol:**

  - Sticky footer integration opening a deeply structured dashboard layout detailing alignment logs, ethical protocols, and automated mitigation charts.


 3. Audio Pipeline Architecture

- Powered entirely by client-side browser **HTML5 Audio / Web Audio API Oscillators**.

- Emits high-frequency clicks during navigation, low-frequency rumble sweeps for background data decryption, and ascending synthetic victory chords upon task completion with zero external audio assets or dependencies.

## 🛠️ Tech Stack

- **Frontend Core:** React.js, TypeScript, Vite

- **Styling:** Tailwind CSS, Lucid Icons

- **Deployment Automation:** GitHub Actions (CI/CD Pipeline)
